docker各组件简介

docker相关的组件有：

  • docker：docker的命令行客户端。
  • dockerd：docker server daemon程序。
  • docker-containerd：高性能容器运行时。等同于containerd。
  • docker-containerd-ctr：docker-containerd的命令行客户端。
  • docker-containerd-shim：垫片组件。解耦containerd和runc，containerd通过shim和runc通信。
  • docker-proxy：容器代理。提供docker容器的端口映射功能。
  • docker-runc：运行容器的命令行工具。等同于runc。

dockerd的进程结构

dockerd的进程模型如下：

dockerd -> docker-containerd ---> docker-containerd-shim -> docker-runc -> runc init -> process
                      			|-- > docker-containerd-shim -> docker-runc -> runc init -> process
                      			+-- > docker-containerd-shim -> docker-runc -> runc init -> process

可以通过pstree查看：

func (r *remote) runContainerdDaemon() error {
  // ...

	args := []string{
		"-l", fmt.Sprintf("unix://%s", r.rpcAddr),

func daemon(context *cli.Context) error {

  supervisor.New(
		context.String("state-dir"),
		context.String("runtime"),
		context.String("shim"),
		context.StringSlice("runtime-args"),
		context.Duration("start-timeout"),
		context.Int("retain-count"))
